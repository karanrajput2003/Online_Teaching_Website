To use project First Download Both Folders

`1. Fronted Folder`
    a. cd Frontend
    b. npm install
    c. npm run dev

`2. Backend Folder`
    a. cd Backend
    b. npm install
    c. npm run dev


Note: Make sure to add own mongodb database url in Backend/index.js
